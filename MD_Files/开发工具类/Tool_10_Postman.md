# 1、Postman 简介

- Postman 是一个很流行的 API 调试工具
- Postman 早期时作为 chrome 的插件
- Postman 现在提供了独立的安装包，同时支持 MAC,Windows,Linux。



## 1.1 Postman 的安装

- 下载地址：https://www.postman.com/downloads/
- 安装完之后，最好注册登录一下，这样，测试过的接口就会有记录。
